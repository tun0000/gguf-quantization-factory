# 實驗:用繁體中文語料校正 imatrix,對繁中品質有沒有幫助?

## 動機

之前的 imatrix 實驗([summary-qwen2.5-3b-instruct.md](summary-qwen2.5-3b-instruct.md))用英文
wikitext-2 校正,PPL 明顯改善。但校正語料是英文,量化的模型主要拿來回答繁中問題——
**校正語料的語言重要嗎?** 這個實驗直接測。

## 方法

- **Calibration 語料**:`wikimedia/wikipedia`(zh 版)串流下載前段文章,以
  [OpenCC](https://github.com/BYVoid/OpenCC) `s2twp`(簡體→台灣正體,含慣用詞轉換如
  软件→軟體)強制轉為繁體,取 400 萬字元([scripts/prepare_zh_corpus.py](../scripts/prepare_zh_corpus.py))
- **PPL 測試語料**:同一資料集**跳過前 2 萬篇文章**後再取 40 萬字元,與校正語料
  **零重疊**,避免校正即考題的資料污染
- 對照組:F16、Q8_0、Q5_K_M、Q4_K_M(無校正)、Q4_K_M / IQ4_XS(英文 wikitext imatrix)
- 實驗組:Q4_K_M / IQ4_XS(繁中維基 imatrix)
- 每個版本都同時跑**繁中 PPL**(新)與**英文 wikitext-2 PPL**(對照,確認繁中校正
  沒有犧牲原本的英文能力)

## 結果:繁中 held-out 語料 PPL

| 量化 | 繁中 PPL ↓ | vs F16 |
|---|---|---|
| F16 | 10.9953 | 基準 |
| Q8_0 | 11.0149 | +0.18% |
| Q5_K_M | 11.1676 | +1.57% |
| Q4_K_M(無校正) | 11.6141 | +5.63% |
| Q4_K_M + 英文imatrix | 11.2747 | +2.54% |
| **Q4_K_M + 繁中imatrix** | **11.2578** | **+2.39%** |
| IQ4_XS + 英文imatrix | 11.4075 | +3.75% |
| **IQ4_XS + 繁中imatrix** | **11.3674** | **+3.38%** |

## 對照:英文 wikitext-2 PPL(確認沒有犧牲通用能力)

| 量化 | 英文 PPL ↓ | vs F16 |
|---|---|---|
| F16 | 9.0631 | 基準 |
| Q4_K_M + 英文imatrix | 9.3241 | +2.88% |
| Q4_K_M + 繁中imatrix | 9.2671 | +2.25% |
| IQ4_XS + 英文imatrix | 9.3713 | +3.40% |
| IQ4_XS + 繁中imatrix | 9.3731 | +3.42% |

速度與 VRAM 峰值(Q4_K_M-ZHTW pp512 20,525 tok/s、tg128 287.5 tok/s、VRAM 4,295 MiB;
IQ4_XS-ZHTW pp512 22,988 tok/s、tg128 292.7 tok/s、VRAM 4,120 MiB)與英文校正版本
在誤差範圍內一致——量化格式相同,imatrix 只影響離線量化時的權重取捨,不影響推論運算量。

## 結論

1. **繁中校正在繁中考題上確實小幅勝出**(Q4_K_M:+2.39% vs +2.54%;IQ4_XS:+3.38% vs
   +3.75%),方向一致但差距不大(0.15~0.37 個百分點),部分落在標準誤差邊緣。
2. **繁中校正沒有犧牲英文能力**——兩個語言的 PPL 同步改善或持平,不是「顧此失彼」的取捨。
3. **語言匹配的邊際效益遠小於「有沒有做 imatrix 校正」本身**:無校正→有校正(英文或
   繁中皆可)把 PPL 損失砍了一半以上(+5.63% → +2.4~2.5%);而校正語料語言只小幅影響最後
   一段路(+2.54% → +2.39%)。
4. **imatrix 修不了模型本身的語言傾向**:Ollama 實測時,繁中校正版本仍然吐出「由**于**」
   這種簡體用詞(見 [ollama_test.md](ollama_test.md))。imatrix 決定量化時哪些權重該保留
   更高精度,不會改寫模型在預訓練階段學到的詞彙分佈——那是模型本身的能力邊界,不是
   量化 pipeline 能解決的問題。

**實務建議**:做 imatrix 校正時,語料的**存在**比語料的**語言**重要得多;如果剛好有目標
語言的語料,用它有微幅加分,但不必為了「語言要匹配」而特地去準備——英文 wikitext-2
校正對繁中任務仍然有效。
